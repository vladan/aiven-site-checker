chweb.collector¶

Checks status of web servers and sends them to a configured Kafka topic.

class chweb.collector.Collector(config: chweb.models.Config, logger: logging.Logger, event_loop: asyncio.events.AbstractEventLoop, queue: asyncio.queues.Queue)¶

A class that contains all methods needed to check the statuses of all websites present in the config.

async check(url: str, regex: Optional[str]) → chweb.models.Check¶

Checks the status of a website and optionally matches a regex on the response body.

Parameters

  • url – The URL of the site that needs to be checked.

  • regex – An optional regex to match on the response body.

Returns

chweb.models.Check, sent to a queue in chweb.collector.Collector.check_forever.

async check_forever(site: chweb.models.SiteConfig)¶

A void function that checks the status of a site and sends the result in an asyncio.Queue for further processing, i.e. the check info is sent to a Kafka topic in chweb.collector.Producer.produce() (as in produce data for the Kafka consumers, defined in chweb.consumer.Consumer.consume.

Parameters

site – A chweb.models.SiteConfig object from the config.

class chweb.collector.Producer(config: chweb.models.Config, logger: logging.Logger, event_loop: asyncio.events.AbstractEventLoop, queue: asyncio.queues.Queue)¶

Kafka producer.

Reads checks from the queue written by chweb.collector.Collector and sends all messages in a kafka topic.

async produce()¶

Creates and starts an aiokafka.AIOKafkaProducer and runs a loop that reads from the queue and sends the messages to the topic defined in the config.

chweb.consumer¶

Sample consumer.

class chweb.consumer.Consumer(config: chweb.models.Config, logger: logging.Logger, event_loop: asyncio.events.AbstractEventLoop, queue: asyncio.queues.Queue)¶

Consumes messages from the kafka topic and if they are correct, i.e. if they can be serialized in a chweb.models.Check object, then they are saved in the database.

async consume()¶

Consumes messages from a kafka topic and writes them in the database.

class chweb.consumer.DbWriter(config: chweb.models.Config, logger: logging.Logger, event_loop: asyncio.events.AbstractEventLoop, queue: asyncio.queues.Queue)¶

Database operations and handy helpers.

async connect()¶

Connects to the database and stores the connection in self.conn.

async setup()¶

Setup the database, i.e. create the table and set up the indexes.

async write()¶

Writes a batch of records to the databse. The size in the batch is defined in chweb.models.PostgresConfig.batch_size.
